# Sri Lutfiya Dwiyeni — Portfolio Website

Personal portfolio website built with React + Vite.

🌐 **Live:** https://rilufiyy.github.io/Web-Portofolio-Sri-Lutfiya-Dwiyeni/

## Tech Stack

- React 19 + Vite
- Pure CSS (no CSS-in-JS)
- react-icons
- Deployed via GitHub Actions → GitHub Pages

## Run Locally

```bash
npm install
npm run dev
```
